핵심 기능

한-일 노선에 특화된 모니터링 파이프라인

🔍

자동 스케줄 수집

14개 항공사 공식 사이트에서 스케줄을 주기적으로 수집. 공항공사 데이터와 교차 검증.

⚡

변경 감지 엔진

신규 취항, 단항/운휴, 빈도 변경, 시간 변경, 기재 변경, 시즌 스케줄까지 8종 이벤트 감지.

🔔

멀티채널 알림

Telegram Bot, Discord Bot으로 실시간 알림. 항공사·노선별 구독 필터링 지원.

📊

웹 대시보드

전체 노선 현황, 변경 타임라인, 항공사별 필터링을 한눈에 볼 수 있는 대시보드.

🗄️

히스토리 추적

모든 변경 이벤트를 DB에 기록. 계절별 패턴 분석, 노선 이력 조회 가능.

🔧

CLI 도구

커맨드라인으로 수집·감지·알림을 제어. 자동화 스크립트와 쉽게 연동.
